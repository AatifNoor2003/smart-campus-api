# Smart Campus API

JAX-RS RESTful API for university Smart Campus coursework. This API manages rooms and sensors for a smart campus system.

## Technology Stack
- **JAX-RS** (Jersey 2.41)
- **Apache Tomcat 9**
- **Java 17**
- **Maven**

## Build & Run Instructions

### Prerequisites
- JDK 17 or later
- Apache Tomcat 9 installed
- Maven (included with NetBeans)

### Build the Project
```bash
mvn clean package
